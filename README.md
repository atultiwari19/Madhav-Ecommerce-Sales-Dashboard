# Madhav-Ecommerce-Sales-Dashboard
* Imported sales data (Orders & Details) from CSV files into Power BI.
* Cleaned and transformed data using Power Query (handled missing values, changed data types, removed duplicates).
* Created custom columns and performed data aggregation using Group By.
* Built data model by establishing relationships between tables using Order ID.
* Designed an interactive dashboard layout with multiple visuals.
* Developed charts (stacked bar, donut) to analyze profit, categories, and payment modes.
* Created DAX measures such as Average Order Value (AOV).
* Added slicers (Quarter, State) for interactive filtering.
* Applied conditional formatting to highlight profit and loss trends.
